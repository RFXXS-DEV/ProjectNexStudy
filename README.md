# NexStudy

> Organize seus estudos em um único lugar.

NexStudy é uma aplicação web desenvolvida para demonstrar conhecimentos em HTML, Tailwind CSS, JavaScript, Python e Flask. O projeto reúne gerenciamento de disciplinas, cálculo de médias, Timer, Pomodoro, metas e estatísticas em uma interface moderna e intuitiva, planejada para evoluir com novas funcionalidades em futuras versões.

---

## Funcionalidades

### Usuários
- Cadastro de usuário
- Login
- Logout

### Dashboard
- Visão geral do desempenho
- Estatísticas de estudo
- Continue estudando
- Continue vendo
- Calendário
- Gráfico de horas estudadas

### Disciplinas
- Criar disciplina
- Editar disciplina
- Excluir disciplina
- Pesquisa de disciplinas
- Cadastro de notas
- Cálculo automático da média

### Estudos
- Timer personalizado
- Pomodoro (25/5)
- Pomodoro (50/10)
- Histórico de estudos
- Registro de horas estudadas

### Configurações
- Tema claro e escuro
- Sistema escolar (bimestral, trimestral e semestral)
- Média para aprovação
- Meta mensal de estudos

## Objetivo

O NexStudy foi desenvolvido com o objetivo de aplicar conhecimentos em desenvolvimento web através da criação de uma aplicação completa, simulando um produto real voltado para organização dos estudos e acompanhamento do desempenho acadêmico.
